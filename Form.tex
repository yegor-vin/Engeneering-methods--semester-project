\section*{Form VV-A: Request for Applied Research}
\textbf{Project Title:} Open-Source, Privacy-First Transit Route Planner with Offline Support\\
\textbf{Applicant:} \textit{Public Transport Development Team}\\
\textbf{Project Type:} Applied Research\\
\textbf{Keywords:} Open-source, Public Transit, Offline Routing, Privacy, Mobile Platforms\\
\textbf{Summary:} This project proposes the creation of an open-source public transit application capable of detailed route pre-planning with offline functionality, ensuring data privacy and transparency.

\section*{Form VV-B: Project Objectives and Outputs}
\textbf{General Objective:} Develop a cross-platform, open-source transit route planner emphasizing privacy and offline functionality.\\
\textbf{Specific Objectives:}
\begin{itemize}
  \item Implement detailed route customization (transfers, timing, distance, preferences).
  \item Ensure offline operation with preloaded data.
  \item Provide transparent, non-commercial data handling.
  \item Support extended platforms (Android, iOS, postmarketOS, KaiOS).
\end{itemize}

\textbf{Measurable Indicators:}
\begin{itemize}
  \item Functional prototype for at least 3 platforms.
  \item Achieve 95\% routing accuracy vs. online reference.
  \item 0 collection of personal user data (verified through audit).
  \item Publication of code under open license with community contributions.
\end{itemize}

\section*{Form VV-D: Project Schedule and Milestones}

\textbf{Milestones:}
